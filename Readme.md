## Setup
- Install vim-plug
- Install nerd font
- Install latest nvim
- Clone this repo and cp it too `~/.config/nvim`
- Open nvim and :PlugInstall
- Restart nvim
- Set true color in term (skip if the color is already right)
- Install LSPs: pyright, biome, lua_ls, tsserver, gopls, zls, clangd, copilot
